Mini Drone Project Version-02

A fully programmable mini drone built from scratch with improved power supply and performance

Project Overview

Drones are always fascinating things. Everyone loves seeing it flying, capturing videos, following them etc. But building one is always difficult. If it is from scratch, then it is more difficult. Lots of skills like- PCB designing, programming, some sorts of algorithm like-PID, Sensor data Filtering are essential to learn. Also, it needs lots of money for building a big one.

So, I decided to make a mini drone, which is low cost and also fully programmable. In the previous version, I was having some issues with the power supply. In this newer version, I have solved the issue by replacing the booster with a LDO and by using a high C-rated battery.

Now I am going to share How I build this drone from scratch.

Battery Selection

While choosing battery one must consider its C rating and weight. C rating is basically the discharge rate of the battery. The motor we are using are really very power hungry. The 4 motors, when these start, suddenly draw a huge amount of current from the battery, almost greater than 20A. If the battery is not able to generate that much power, then the microcontroller will reset again and again. Also, in full load the drone takes almost 12 A continuous current. So, to ensure the power we need to know the C rating of the battery.

For example, currently I am using the battery shown below. It shows C rating of 80C and 300mah. So, it can deliver maximum up-to 0.300*80 or 24A continuously and weighs only 10g, which satisfy our requirements.

Motor Connection with ESP32

Below is a detailed table for motor connection:

ESP32 Pin Motor No. Direction
271 (front right)CCW (Counter Clock-Wise)
252 (rear right)CW (Clock-Wise)
333 (rear left)CCW
324 (front left)CW

If you prefer building your own PCB, you should consider the dot on the MPU6050 is pointing toward the left rear of the drone. Otherwise, the drone will flip upside down.

Software Setup

I guess these are the least steps to complete the hardware setup. Now we can move toward the software setup. Again, download the software and remote apk file from the same GitHub link. Run the code using Arduino IDE in the esp32 and install the apk in the android. Make sure your Arduino IDE is ESP32 compatible. Connect the battery to the drone and reset and keep it still for at least 30s, so that the calibration process in done. Then look for a hotspot name "drony" in the Wi-Fi checklist. Connect it using the password 11223344, then open the app. Click on pair, ARM and you are ready to fly.
